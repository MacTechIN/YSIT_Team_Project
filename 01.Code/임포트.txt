Streamlit 설치 및 사용법:

설치: Streamlit은 다음과 같이 터미널 또는 명령 프롬프트에서 설치할 수 있습니다.
Copy code
pip install streamlit
사용법: Streamlit은 Python으로 웹 애플리케이션을 만들기 위한 프레임워크입니다. 일반적으로 스크립트(예: app.py)를 작성하고 다음 명령어를 사용하여 실행합니다.
arduino
Copy code
streamlit run app.py
스크립트(app.py)에서는 st를 사용하여 다양한 UI 요소를 만듭니다. 예를 들어, st.title()은 제목을 설정하고, st.write()는 텍스트를 표시합니다.
Pandas 및 NumPy:

설치: Pandas와 NumPy는 일반적으로 아나콘다와 같은 과학 계산 라이브러리를 설치할 때 포함됩니다. 별도로 설치해야 하는 경우 다음 명령어를 사용할 수 있습니다.
Copy code
pip install pandas numpy
사용법: Pandas는 데이터 조작 및 분석에 사용되며, NumPy는 대규모 다차원 배열 및 행렬 지원에 사용됩니다. Pandas와 NumPy의 공식 웹사이트에서 상세한 문서를 찾을 수 있습니다.
CatBoost:

설치: CatBoost는 다음과 같이 설치할 수 있습니다.
Copy code
pip install catboost
사용법: CatBoost는 결정 트리를 기반으로 하는 그래디언트 부스팅을 위한 머신 러닝 라이브러리입니다. 회귀 또는 분류 작업에 사용할 수 있으며, 회귀 작업에는 CatBoostRegressor 클래스가 사용됩니다.
Joblib:

설치: Joblib는 다음과 같이 설치할 수 있습니다.
Copy code
pip install joblib
사용법: Joblib은 파이썬에서 가벼운 파이프라인을 구축하기 위해 사용됩니다. 특히 큰 NumPy 배열을 효율적으로 저장하고 로드하는 데 유용합니다.
Time 및 TQDM:

이들은 표준 파이썬 모듈입니다. 별도의 설치가 필요하지 않습니다.
time은 시간을 추적하기 위해 사용되며, tqdm은 반복 작업에 대한 진행률 표시를 제공합니다.


Warnings:

설치: Warnings 모듈은 파이썬에 기본적으로 포함되어 있습니다.
사용법: Warnings 모듈은 경고 메시지를 처리하는 데 사용됩니다. 경고를 무시하거나 특정 작업에서 경고 메시지를 출력하지 않도록 설정할 수 있습니다.
Streamlit Folium:

설치: Streamlit Folium은 다음과 같이 설치할 수 있습니다.
Copy code
pip install streamlit-folium
사용법: Streamlit Folium은 Streamlit 애플리케이션에서 Folium 지도를 사용하기 위한 라이브러리입니다. folium_static 모듈은 Folium을 Streamlit 앱에서 사용할 수 있도록 도와줍니다.
Folium:

설치: Folium은 다음과 같이 설치할 수 있습니다.
Copy code
pip install folium
사용법: Folium은 지리적 데이터를 시각화하기 위한 Python 라이브러리입니다. 지도를 만들고 다양한 마커, 도형, 팝업 등을 추가하여 상호작용성 있는 지도를 생성할 수 있습니다.
